Two-colour microarray experiments

Annotare currently only supports one type of two-colour microarray workflow (see below). For other workflows, please continue to use Annotare to construct the submission (e.g. experiment background, protocols, sample description, assigning files to labeled extracts).

When you get to the stage where you have validation errors about "[Assign Files] The assignment between files and samples contains errors.", please write us using the "Contact Us" button from within your submission form and a curator will assist. We will be supporting more workflows in the next few months.
